# ⚡ Smart Power Monitor

A modern and responsive web application that helps users monitor household electricity usage, analyze appliance-wise power consumption, visualize energy data, and receive smart energy-saving recommendations.

---

## 📌 Project Overview

Smart Power Monitor is designed to help homeowners understand how electricity is consumed by different appliances. The application provides a professional dashboard with energy statistics, appliance monitoring, monthly reports, and energy-saving tips.

---

## ✨ Features

- 🏠 Modern Landing Page
- 🔐 User Login Interface
- 📊 Interactive Dashboard
- ⚡ Energy Consumption Monitoring
- 🔌 Appliance-wise Power Usage
- 💡 Smart Energy Saving Suggestions
- 📈 Monthly Reports
- 📉 Interactive Charts (Chart.js)
- 📱 Fully Responsive Design
- 🌙 Clean & Modern UI
